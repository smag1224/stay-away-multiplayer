# Нечто — Stay Away!

Веб-реализация настольной карточной игры «Нечто из глубокой бездны» (Stay Away!).
Мультиплеер для 4–12 игроков через браузер — каждый игрок на своём устройстве.

## Запуск

```bash
npm install

# Запустить сервер и клиент в двух терминалах:
npm run dev:server   # backend на порту 8787
npm run dev          # frontend на порту 5173
```

Откройте `http://localhost:5173` в браузере.

## Правила (кратко)

**Цель:** Один игрок — **Нечто** (The Thing). Остальные — **Люди**. Нечто пытается заразить или уничтожить всех людей. Люди пытаются найти и сжечь Нечто Огнемётом.

**Ход (3 шага):**
1. **Взять карту** — из колоды. Паника играется сразу. Событие добавляется в руку, затем нужно сыграть или сбросить 1 карту.
2. **Обмен** — передать 1 карту соседу по направлению хода (тот обязан дать свою взамен). Защитные карты могут отменить обмен.
3. **Конец хода** — ход переходит к следующему игроку.

**Роли:**
- **Нечто** — единственный, кто может заражать (передавая карту «Заражение!» при обмене)
- **Человек** — ищет Нечто, может использовать Огнемёт
- **Заражённый** — помогает Нечто тайно

**Победа:**
- Люди: уничтожить Нечто Огнемётом
- Нечто: заразить или уничтожить всех людей, затем объявить победу

## Технологии

- **Frontend:** React + TypeScript + Vite, Framer Motion, двуязычный интерфейс (RU/EN), mobile-first
- **Backend:** Node.js HTTP сервер с SSE для синхронизации состояния в реальном времени
- **Тесты:** Vitest (35 тестов)
